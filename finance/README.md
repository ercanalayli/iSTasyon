# AperiON Finans Modulu

Bu klasor AperiON finans ve kar zekasi modulu icindir.

Son surum: AperiON_Finans_Panel_v33.html

Kapsam:
- Dinamik kar motoru
- Hizmet / mal / karma satis ayrimi
- Hizmette mal maliyeti 0
- Mal satisinda urun maliyeti
- Dinamik gider payi
- Sabit gider kartlari
- Degisken gider merkezi
- Gun sonu kar kapanisi
- Rapor dagitimi
- Telegram AperiON BizimHesap koordinasyonu
- Urun hizmet siniflandirma
- Kar guven skoru

Not: Ana index.html bu adimda degistirilmedi. Once finans modulu bagimsiz dosya olarak eklendi.
